<p align="center"> 极简的git账号管理工具（Git Account Management Tools ）</p>

### Features

- ✨ git账户无痛切换

- ⚡️ 简便的cli 工具

- 🚀 极简的操作方式

- 😊 gacm ls 自动添加本地git账户

- 💀 更新会丢失用户


### Getting Started


```shell
## just like nvm

gacm ls


gacm use xxx ---local

gacm use xxx ---global

gacm add --name xxx --email xxx

gacm delate user

```
